# xerweon™ GATEKEEPER

Official releases repository for xerweon™ GATEKEEPER.

> **Note:** This repository contains only releases. Source code is maintained in a private repository.

## Download

Download the latest version from the [**Releases**](https://github.com/Xerweon/gatekeeper/releases/latest) page.

### Supported Platforms

- **Windows** (x64) - `.msi` installer
- **macOS** (Intel & Apple Silicon) - `.dmg` installer  
- **Linux** - `.AppImage`, `.deb`, `.rpm` packages

## Installation

### Windows
Download the `.msi` file and run the installer.

### macOS
1. Download the `.dmg` file for your architecture
2. Open and drag GATEKEEPER to Applications

### Linux
Download and run the `.AppImage`, or install the `.deb`/`.rpm` package.

## Automatic Updates

GATEKEEPER includes built-in automatic updates.
